# 🎞️ GIF Maker App (Python Tkinter)

A simple desktop application built using **Python Tkinter** that allows users to select multiple images and convert them into an animated GIF. The app includes image preview, adjustable speed control, and progress tracking.

---

## 🚀 Features

- 🖼️ Select multiple images (PNG, JPG, JPEG, BMP)
- 👀 Preview selected images as thumbnails
- 🎬 Convert images into an animated GIF
- 🎚️ Adjust GIF speed (milliseconds per frame)
- 📊 Progress bar during GIF creation
- 💾 Save GIF to local storage
- ⚡ Smooth UI using threading

---

## 🛠️ Technologies Used

- Python 🐍
- Tkinter (GUI Development)
- Pillow (PIL - Image Processing)
- Threading (for background task execution)
- OS module (file handling)

---

## 📦 Installation

### 1. Clone the repository
```bash
git clone https://github.com/hasmath27/gif-maker-app.git
````

### 2. Move into project folder

```bash
cd gif-maker-app
```

### 3. Install required library

```bash
pip install pillow
```

---

## ▶️ How to Run

Run the project using:

```bash
python gif_maker_app.py
```

---

## 🖥️ How to Use

1. Open the application
2. Click **🖼️ Select Images**
3. Choose multiple images from your device
4. Preview images in the app
5. Adjust GIF speed using the slider
6. Click **🎬 Create GIF**
7. Choose location to save your GIF file
